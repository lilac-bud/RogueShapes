# Rogue Shapes

Проект был разработан как часть прохождения туториала [Roguelike 2D](https://learn.unity.com/course/2d-roguelike-tutorial).

В процессе разработки мне пришлось решить множество проблем, неучтённых авторами туториала. В связи с многочисленными внесёнными изменениями в предлагаемый код программы, а также использованием собственных спрайтов я решила выложить игру на гитхаб.

***

This project has been developed as part of going through [Roguelike 2D](https://learn.unity.com/course/2d-roguelike-tutorial) tutorial.

During development I had to solve a lot of problems not accounted for by the tutorial authors. Due to numerous changes implemented in the proposed code and usage of my own sprites, I decided to post this game on github.
