# Rogue Shapes

<img width="1920" height="1080" alt="rogue_shapes" src="https://github.com/user-attachments/assets/35f79888-6497-4379-b1f5-0d1b147ab319" />

Проект был разработан как часть прохождения туториала [Roguelike 2D](https://learn.unity.com/course/2d-roguelike-tutorial).

В процессе разработки мне пришлось решить множество проблем, неучтённых авторами туториала. В связи с многочисленными внесёнными изменениями в предлагаемый код программы, а также использованием собственных спрайтов я решила выложить игру на гитхаб. В игру можно поиграть [здесь](https://play.unity.com/en/games/89a1ed72-5b19-4ec6-8a79-cfcca0c9add0/rogue-shapes).

***

This project has been developed as part of going through [Roguelike 2D](https://learn.unity.com/course/2d-roguelike-tutorial) tutorial.

During development I had to solve a lot of problems not accounted for by the tutorial authors. Due to numerous changes implemented in the proposed code and usage of my own sprites, I decided to post this game on github. The game can be played [here](https://play.unity.com/en/games/89a1ed72-5b19-4ec6-8a79-cfcca0c9add0/rogue-shapes).
